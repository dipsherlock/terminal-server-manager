# TSM - Terminal Server Manager
A menu-driven TUI for managing a home server.

Built by Skyler Clark using: 
- Python
- Textual (Python API)

## TSM UI Framework
- BRAIN: Midnight Commander (mc)
    - file/media management
- HEART: Network Manager Text User Interface (nmtui)
    - network management
- VITALS: Btop (btop++)
    - resource monitoring
